Canonn API Changelog

v1.1.0

Key features: API Auth & docs.

  • Switched to Auth0 API authorization schema
  • Added swagger based automated API documentation
  • Added EDSM coordinates to systems with code for updates on a regular basis
  • Moved secondary artifact from obelisk to codex entry
  • Extended map endpoints to return more info
  • Added verified flag to obelisks
  • Added this about page ;-)
  • Small UI tweaks (i.e. better positioned edit buttons)
  • Dev, test and prod environments now have different color themes
  • Updated dependencies to latest
  • Fixed potential hung database updates

v1.0.0

Initial version with basic features.

  • Show base data entries for all users
  • Auth0 authentication
  • Provide edit (basic CRUD) operations for base data entries for users with permissions
  • Automatic deployment and automatic database upgrades on deployment
  • Sync of system and body id's with EDSM entry id's
