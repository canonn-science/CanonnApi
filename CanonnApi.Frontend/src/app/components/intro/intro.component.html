Welcome to Canonn API

This is the Canonn Science API system. The API will allow you to easily consume the data that Canonn.science gathers during its research.

General information: If you just want to browse the data, you do not need to sign in.
Unless you know you should be able to edit the data, and have been granted permission to do so (ask the Canonn Science R&D team for that), there will be no difference for you if you sign in.

Geeky / technical information: This administration interface is build with Angular 4.0. The API backend is build on top of .NET Core. You can find the source at Github

API documentation

You can find the api documentation in an explorable way here on our API documentation website.

We are using swagger to provide you with all information you need about our API, so you can use the swagger ui to learn and experiment with our API. You can also use the swagger files shown there to automatically create clients for our API.

  Contact information

You can easily contact the Canonn R&D team on the Canonn Science Discord chat. We are happy to help you with any questions regarding the Canonn API system. Ask for one of the R&D team members.

If you need technical help with this system, find a bug or have a feature request, feel free to create an issue on our GitHub page.

If you feel like you should have write access to the Canonn Science API system, please contact one of our R&D team members to set you up for access. To speed things up, first sign in with your Google or Twitter account and then contact us providing your Google account email address or Twitter handle.
