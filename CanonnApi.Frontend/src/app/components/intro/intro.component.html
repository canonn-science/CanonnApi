Welcome to Canonn API

This is the Canonn API. The API will allow you to easily consume the data that Canonn.science gathers during its research.

Important information: There is no need to sign in if you just want to browse the data. Unless you know you should be able to edit the data, and have been granted permission to do so (ask the Canonn R&D team for that), there will be no difference for you even if you sign in.

This is work in progress. And we need to come up with a bit more content ;-)

Geeky information: This administration interface is build with Angular 4.0 beta. The API is build on top of .NET Core. You can find the source at Github
